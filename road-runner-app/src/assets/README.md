
# Assets Guide

This folder is reserved for your project assets (images, icons).

## Placeholders
- [ ] **app_icon.png**: App Icon (Must be 512x512px PNG for Base44 deployment).
- [ ] **marker_icon.png**: Map marker icon (e.g., blue dot or arrow arrow) to indicate current location.
- [ ] **temp_logo.png**: Placeholder for the app logo.
- [ ] **map_overlay_bg.png**: Optional background overlays for the UI.

Simply overwrite these files with your actual assets to update the app appearance instantly.
